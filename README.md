# SI_TP2

## Taille du GL Canvas

La taille du canvas se règle dans le fichier `glstyle.css`.

```css
#oglcanvas {
    width: 70%;
    height: 70%;
    display: block;
    border: none;
}
```

## Controls

1. Shaders Menu

   Pour faire apparaître et disparaître le menu des shaders, appuyez sur la touche _suppr_ du pavé numérique.
